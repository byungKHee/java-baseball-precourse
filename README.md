# java-baseball-precourse
## 기능 목록
### 1. 게임 초기화
- [ ] 1~9 사이의 서로 다른 숫자 N개 생성
- [ ] N 값은 상수로 정의
- [ ] 정답/입력에 대한 객체 정의
### 2. 사용자 입력
- [ ] N개의 숫자 입력 문자열로 받기
- [ ] 입력 값 유효성 검사
    - N개의 연속된 숫자 문자열로 입력 ex) "123" (O), "479" (O), "112" (X), "12a" (X), "123 4" (X)
    - 사용자가 잘못된 값을 입력할 경우 [ERROR]
### 3. 정답 판단
- [ ] 스트라이크 - 같은 위치에 같은 숫자
- [ ] 볼 - 다른 위치에 같은 숫자
- [ ] 낫싱 - 일치하는 숫자 없음
- [ ] 판정 결과에 대한 객체 정의
### 4. 결과 출력
- [ ] 판정 결과 객체를 문자열로 변환하여 출력
    - ex) "1스트라이크 2볼", "낫싱"
### 4. 게임 종료 및 재시작
- [ ] 3스트라이크 시 게임 종료
- [ ] 게임 종료 후 재시작 여부 확인

## 설계 원칙
### 1. MVC 패턴
- Model: 게임의 상태와 로직을 관리하는 클래스들 (예: 숫자, 판정 결과)
- View: 사용자와의 상호작용을 담당하는 클래스들 (예: 입력 및 출력)
- Controller: Model과 View를 연결하고 게임의 흐름을 제어하는 클래스들
